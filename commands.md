# Chapter 1

## commands
```shell
# 가상환경 생성
pipenv shell

# package install
pip install django
pip install selenium

# create django project
django-admin.py startproject superlists

# run django server
cd superlists
python manage.py runserver

# package freeze
pip freeze > requirements.txt
```
